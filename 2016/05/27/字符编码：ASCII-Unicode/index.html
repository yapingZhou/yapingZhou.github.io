发表于 2016-05-27   |  

ASCII和Unicode的区别

ASCII码

ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是基于拉丁字母的一套电脑编码系统，最后一次更新是在1986年。至今为止共定义了128个字符，其中33个字符无法显示（一些终端提供了扩展，使得这些字符可显示为诸如笑脸、扑克牌花式等8—bit符号），且这33个字符多数都已是陈废的控制字符。控制字符的用途主要用来操控已经处理过的文字。在33个字符之外的是95个可显示的字符，包含用键盘敲下空白键所产生的空白字符也算1个可显示字符。

ASCII的局限在于只能显示26个基本拉丁字母、阿拉伯数字和英式标点符号，因此只能用于显示现代美国英语。

Unicode

Unicode是计算机科学领域的一项业界标准。它对世界上大部分的文字系统进行了整理、编码，使得电脑可以用更为简单的方式来呈现和处理文字。
Unicode伴随着通用字符集的标准而发展，同时也已书本的形式对外发表。Unicode至今仍在不断增修，版本8.0.0已收入超过十万个字符。

UTF-8：UTF意为通用字集转换格式（Universal Character Set Transformation Format),UTF-8是Unicode的8位元格式。
UTF-8是一种变长的编码方式，字符长度从1个字节到4个字节不等。越是常用的字符，字节越短，最前面个的128个字符，只使用1个字节表示，与ASCII码完全相同。
